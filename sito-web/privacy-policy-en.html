This Privacy Policy describes how the website www.andreapiani.com (hereinafter "Website") collects, uses, and protects users' personal data, in compliance with the General Data Protection Regulation (GDPR - EU Regulation 2016/679).

The Website is a purely static site hosted on Vercel. It has no backend, no database, and no server-side application. It does not perform any analytics or tracking, sets no profiling cookies, and runs no chatbot or artificial-intelligence assistant. The amount of personal data processed is therefore minimal, as described below.

1. Types of Data Collected

The Controller collects the following types of personal data:

1.1 Data Provided Voluntarily by the User (contact forms via WhatsApp)

The Website's contact and quote-request forms collect the data you type — typically:

  • Name and Surname
  • Email
  • Phone Number (optional)
  • Company/Organization (optional)
  • Project Description and other technical information provided in forms

Because the Website has no backend, this information is handled only locally in your browser. When you submit a form, the data is used solely to pre-fill a WhatsApp message; it is transmitted only if you actively send that message from your own WhatsApp app. In that case the recipient is Andrea Piani and the message is delivered through Meta Platforms Ireland Ltd. (WhatsApp), acting as data processor. No form data is received or stored on the Website's servers, which do not exist for this purpose.

1.2 Navigation Data

The Website performs no analytics and no tracking. It does not collect, log, or store visitors' IP addresses, user agents, page-view events, or any behavioural data of its own. No first-party statistics endpoint exists.

The only third party that may receive a technical connection log is the hosting provider, Vercel Inc., which — like any web host — may process the connection IP address and basic request metadata to deliver the pages and ensure security and availability of the service (see §4).

1.3 Cookies and Local Storage

The Website uses:

  • Necessary technical cookies: only cookie_consent to remember your consent preferences (12 months).
  • localStorage: only technical user-interface preferences (for example the dark-mode theme). No tracking identifiers are stored.
  • Profiling or third-party analytical cookies: none.

For more details, see our Cookie Policy.
