1. Introduzione

Benvenuto in macOS PrestaScan. La tua privacy è estremamente importante per noi. Questa Privacy Policy spiega quali dati raccogliamo, come li utilizziamo e quali sono i tuoi diritti in merito ai tuoi dati personali.

📌 Principio Fondamentale: macOS PrestaScan è progettato con la privacy al centro. Tutti i dati vengono archiviati localmente sul tuo Mac. Non raccogliamo, non trasmettiamo e non vendiamo i tuoi dati personali.

2. Raccolta Dati

2.1 Dati Raccolti Localmente

L'applicazione raccoglie e archivia localmente sul tuo Mac i seguenti dati:

  • URL Scansionati: Gli indirizzi web dei siti che scegli di analizzare
  • Risultati Scansione: Dati SEO estratti dalle pagine (title, meta description, H1, canonical, hreflang, ALT text)
  • Issues Rilevati: Problemi SEO identificati durante l'analisi
  • Fix Suggeriti: Suggerimenti di correzione generati dall'applicazione
  • Configurazioni Progetti: Impostazioni di scansione salvate
  • Cronologia Progetti: Lista dei progetti recenti con statistiche

2.2 Dati NON Raccolti

⚠️ Importante: macOS PrestaScan NON raccoglie:
  • ❌ Dati di utilizzo o analytics
  • ❌ Informazioni personali identificabili
  • ❌ Dati di navigazione o comportamento
  • ❌ Informazioni di contatto
  • ❌ Dati di crash o diagnostica (a meno che non siano esplicitamente condivisi da te)
  • ❌ Cookie o tracker di terze parti

3. Utilizzo dei Dati

I dati raccolti vengono utilizzati esclusivamente per:

  1. Funzionalità dell'App: Eseguire scansioni SEO e generare report
  2. Generazione Fix: Creare suggerimenti di correzione basati sui problemi rilevati
  3. Archiviazione Locale: Salvare i risultati per consultazioni future
  4. Export: Permetterti di esportare i dati in formati CSV, JSON, SQL o HTML

4. Archiviazione dei Dati

4.1 Posizione dei Dati

Tutti i dati vengono archiviati localmente sul tuo Mac in:

  • Database SQLite: ~/.pyprestascan/[nome_progetto]/crawl.db
  • Report HTML: ~/.pyprestascan/[nome_progetto]/reports/
  • Configurazioni: UserDefaults di macOS

4.2 Durata Conservazione

I dati vengono conservati indefinitamente sul tuo Mac fino a quando non decidi di eliminarli. Puoi eliminare i dati in qualsiasi momento:

  • Eliminando l'applicazione
  • Eliminando manualmente la cartella ~/.pyprestascan/
  • Eliminando progetti specifici dall'interfaccia dell'app

4.3 Backup

Se utilizzi Time Machine o altri sistemi di backup, i dati dell'applicazione potrebbero essere inclusi nei backup. Sei responsabile della gestione dei tuoi backup.

5. Servizi di Terze Parti

5.1 Servizi AI (Opzionali)

Se scegli di abilitare le funzionalità AI, l'applicazione può inviare dati ai seguenti servizi di terze parti:

🤖 Provider AI Supportati:
  • DeepSeek API (https://deepseek.com)
  • OpenAI API (https://openai.com)
  • Anthropic Claude API (https://anthropic.com)

5.2 Dati Inviati ai Provider AI

Quando utilizzi le funzionalità AI, vengono inviati:

  • URL delle pagine con problemi SEO
  • Tipo di problema rilevato
  • Richiesta di generazione fix
⚠️ Nota Importante:
  • Le funzionalità AI sono completamente opzionali
  • Devi fornire la tua API key personale
  • Le API key sono archiviate localmente sul tuo Mac
  • Non abbiamo accesso alle tue API key
  • Sei responsabile dei costi delle API di terze parti
  • I provider AI hanno le proprie privacy policy che dovresti consultare

5.3 Richieste di Rete

L'applicazione effettua richieste HTTP/HTTPS ai siti web che scegli di scansionare:

  • Le richieste vengono effettuate direttamente dal tuo Mac
  • Non passano attraverso server intermediari
  • Utilizziamo un User-Agent identificabile: macOSPrestaScan/1.0
  • Rispettiamo i file robots.txt (se configurato)

6. Sicurezza

6.1 Misure di Sicurezza

Implementiamo le seguenti misure di sicurezza:

  • App Sandbox: L'applicazione è sandboxed secondo le linee guida Apple
  • Hardened Runtime: Protezione runtime abilitata
  • Archiviazione Locale: Tutti i dati rimangono sul tuo Mac
  • Permessi Limitati: Richiediamo solo i permessi necessari (network, file selezionati)
  • Nessun Server Remoto: Non utilizziamo server per archiviare i tuoi dati

6.2 API Key

Le API key per i servizi AI sono archiviate in modo sicuro utilizzando i meccanismi di sicurezza di macOS (UserDefaults con protezione sandbox). Tuttavia, ti consigliamo di:

  • Utilizzare API key con limiti di spesa configurati
  • Monitorare l'utilizzo delle API sui dashboard dei provider
  • Revocare le API key se non utilizzi più l'applicazione

7. I Tuoi Diritti

Hai i seguenti diritti riguardo ai tuoi dati:

7.1 Diritto di Accesso

Puoi accedere a tutti i tuoi dati in qualsiasi momento attraverso l'interfaccia dell'applicazione o navigando nelle cartelle di archiviazione.

7.2 Diritto di Cancellazione

Puoi eliminare i tuoi dati:

  • Eliminando progetti specifici dall'app
  • Eliminando la cartella ~/.pyprestascan/
  • Disinstallando l'applicazione

7.3 Diritto di Portabilità

Puoi esportare i tuoi dati in formati standard (CSV, JSON, SQL, HTML) utilizzando le funzionalità di export dell'applicazione.

7.4 Diritto di Opposizione

Puoi disabilitare in qualsiasi momento le funzionalità AI o qualsiasi altra funzionalità che richiede l'invio di dati a servizi esterni.

8. Privacy dei Minori

L'applicazione non è destinata a minori di 13 anni. Non raccogliamo consapevolmente dati personali da minori. Se sei un genitore o tutore e ritieni che tuo figlio ci abbia fornito dati personali, ti preghiamo di contattarci.

Ti consigliamo di rivedere periodicamente questa Privacy Policy per eventuali modifiche. Le modifiche a questa Privacy Policy sono effettive quando vengono pubblicate su questa pagina.
