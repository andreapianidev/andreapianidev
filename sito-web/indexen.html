Cybersecurity - Open Source

  • Blacktooth - Bluetooth Deauther
    Blacktooth is a script designed to execute a deauthentication attack on nearby Bluetooth devices. This tool allows advanced users to disconnect specific Bluetooth devices or all devices within a surrounding area. It is important to note that the use of this script may be subject to legal restrictions in your jurisdiction, so it is recommended to use it only for educational or legitimate research purposes. Developed by Andrea Piani.
  • Madeleine - Wifi Cracker and Deauther
    Madeleine WiFi Cracker and Deauther is an advanced tool designed to test the security of wireless networks. This software provides users with the ability to perform cracking operations on protected WiFi networks and execute deauthentication attacks to test the robustness of network configurations and security. Key Features: WiFi Network Cracking: Madeleine WiFi Cracker allows users to perform cracking operations on protected WiFi networks using various techniques, such as brute force or dictionary attacks. Network Security Analysis: Users can use the tool to assess the security of WiFi networks, identifying vulnerabilities and weak points in their configurations. Deauthentication Attacks: Madeleine WiFi Deauther enables users to carry out deauthentication attacks to disconnect specific devices or all devices connected to a WiFi network. Developed by Andrea Piani.
