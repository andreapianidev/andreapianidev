Step 7 · Le domande che mi fanno sempre

10 domande oneste,
10 risposte oneste.

1. Cos'è Perfex CRM e perché lo proponi?
Perfex CRM è un software gestionale clienti self-hosted con licenza una tantum (~€59 sviluppatore, una tantum). Lo propongo quando il cliente vuole una piattaforma seria — pipeline commerciale, ticket, fatturazione, project management, knowledge base — senza pagare canoni mensili e potendo ospitarla sul proprio server in Italia.
2. Perfex è meglio di Salesforce o HubSpot?
Non in assoluto. Salesforce e HubSpot vincono su funzionalità enterprise (AI, integrazioni native con centinaia di servizi, supporto multinazionale, sales engagement automation). Perfex vince su: costo (una tantum vs canone), ownership (codice e dati tuoi, server tuo), personalizzazione profonda. Per la maggior parte delle PMI italiane è la scelta giusta.
3. Posso integrare Perfex col mio gestionale o con SDI?
Sì. Sviluppo moduli custom Perfex per integrazioni con: gestionali contabili (TeamSystem, Zucchetti, Profis), fatturazione SDI, marketplace, email marketing (Mailchimp, Brevo, Klaviyo), telefonia VoIP, calendari (Google/Outlook), Slack/Teams. Perfex ha API decenti e moltissimo si fa via custom hooks.
4. Quanto tempo serve per averlo operativo?
Setup base con dati di prova: 3-5 giorni. Setup completo con import dati esistenti, configurazione pipeline, training team: 2-4 settimane. Implementazioni con moduli custom o integrazioni complesse: 6-10 settimane. Il preventivo arriva scritto entro 48h dopo l'health check.
5. Cosa serve come server?
Un VPS con PHP 8.1+, MySQL 8 / MariaDB 10, mod_rewrite. Consiglio Hetzner o Aruba per la maggior parte dei casi. Per team più grandi (10+ commerciali, integrazioni pesanti) servono macchine più dimensionate. Lo capiamo durante l'audit.
6. E la sicurezza? Posso fidarmi di un CRM self-hosted?
Sì se gestito bene. Lavoro standard: aggiornamenti Perfex tempestivi, firewall applicativo, password policy forti, 2FA obbligatoria per gli utenti admin, backup giornalieri off-site, log monitoring. È più sicuro di una password debole su SaaS in cloud altrui, sicuramente più allineato GDPR.
7. Sviluppi anche moduli completamente nuovi?
Sì. Moduli custom Perfex per: dashboard custom per management, automazioni pipeline avanzate, integrazioni API esterne, sistemi di reporting avanzato, gestione magazzino e prodotti, calendari multi-utente. Codice tuo, conforme alle specifiche tecniche Perfex ufficiali, documentato.
8. Cosa succede se Perfex chiude o non aggiorna più?
Il codice di Perfex è sul tuo server. Anche se il vendor (CodeCanyon) sparisse domani, il tuo shop continua a funzionare. Servirebbe poi una migrazione a un altro stack (SuiteCRM, ERPNext, custom) che possiamo gestire — ma a differenza dei SaaS, non sei costretto a migrare nello stesso istante.
9. Sei un singolo freelance: e se sparisci?
Tutto il codice scritto per la tua installazione è su repository Git intestato a te. Documentazione, manuali, configurazione: tutto consegnato. Qualsiasi altro sviluppatore PHP (Perfex è open-ish nel codice) può continuare il lavoro. Nessuna dipendenza obbligata da me.
10. Posso parlare con un cliente reale prima di firmare?
Sì. Su richiesta ti metto in contatto diretto con uno dei clienti attivi su Perfex (agenzie, studi professionali, software house). Chiamata reale, niente referenze finte, niente case study costruiti. Scrivimi su WhatsApp e te lo organizzo →
